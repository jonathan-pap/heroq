    Px of padding between each piece's icon and the cell wall. Each art set (canonical / alt) keeps its own values — flip Alt furniture art to tune the other set.

    linear (N×1)

    Natural orientations

    Override each piece type's natural source direction. Changes save automatically to data/pieces/furniture-naturals.json on the server, so the live game inherits them on reload.

    • ↑↓←→ — nudge 1 cell

    • Esc — clear selection
    • Ctrl+S — save
    • Ctrl+Z / Ctrl+Shift+Z — undo/redo
